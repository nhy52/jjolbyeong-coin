## 🐞 알려진 이슈 · 기술 부채

- **레거시 데모 API** — `/api/demo/*`(초기 PoC)와 프론트 흔적은 남아 있으나 현재 화면 미사용. 정리 필요.
- **SSE 토큰 전달** — `EventSource` 특성상 access token을 query로 전달(HTTPS 전제). 로그 노출 주의.
- **단일 프로세스 pub/sub** — 다중 워커/인스턴스 확장 시 Redis Pub/Sub으로 교체 필요(기획서 9.3).
- **개발 서버 재기동** — 백엔드 `--reload` 사용 시 고아 워커가 옛 코드를 서빙한 사례 있음. 재시작 시 포트 점유 프로세스 확인 권장.
- **미결정 정책** — 대장 코인 지급/차감 정책(벌점성 차감)은 기획 보류 상태.
